Binary Method:
    - Network/Subnet address:
        - Fill the host portion of an address with binary O's
    - Broadcast Address:
        - Fill the host portion of an address with binary 1'see
    - First Host:
        - Fill the host portion of an address with binary 0's except for the last bit which is set to binary 1
    - Last Host:
        - Fill the host portion of an address with binary 1's except for the last bit which is set to binary 0
    - Examples:
        - First Example :
        192.168.1.18/24 or 192.168.1.18 255.255.255.0

        Subnet = 192.168.1.00000000 or 192.168.1.0
        1st Host = 192.168.1.00000001 or 192.168.1.1
        Last Host = 192.168.1.11111110 or 192.168.1.254
        Broadcast = 192.168.1.11111111 or 192.168.1.255

    - Second Example   
        172.16.35.123/20 or 172.16.35.123 255.255.240.0

        172.16.0010 0011.01111011

        Subnet = 172.16.00100000.00000000 or 172.16.32.0
        Host 1 = 172.16.00100000.00000001 or 
        172.16.32.1 
        Host 2 = 172.16.00101111.11111110 or 
        172.16.47.254
        Broadcast = 172.16.00101111.11111111 or 172.16.47.255

    - Third Example   
        172.16.129.1/17 or 172.16.129.1 255.255.128.0

        172.16.1 0000001.00000001
        Subnet = 172.16.10000000.00000000 or 172.16.128.0
        Host 1 = 172.16.100000000.00000001 or 
        172.16.128.1
        Host 2 = 172.16.11111111.11111110 or 
        172.16.255.254
        Broadcast = 172.16.11111111.11111111 or 172.16.255.255
